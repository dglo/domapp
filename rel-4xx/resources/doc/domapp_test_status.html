TEST STATUS SUMMARY

Testing of the first beta releases have been under way since June 1, using the DMT test software, using the PSL Cable Test System (PCTS) and two hardware configurations at LBNL.  The list of tests on that page gives an idea of what is currently being tested.

SUMMARY OF RECENT TEST RESULTS

Two 24-hour tests were run on 10 DOMs on PCTS.  There were two failures where the SPE rate dropped below the expected number.  Assuming the failures are representative of the average [it is curious that they occur on the same DOM], the test failure rate per DOM is something like 1/10 days.  No catastrophic errors (DOM crashes or loss of communications) were observed.

It's currently under investigation whether the test failures shown are due to problems in the testing software or in domapp per se.
