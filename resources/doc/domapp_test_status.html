TEST STATUS SUMMARY

Testing of the first and second beta release have been under way since June 1, using the DMT test software, using the PSL Cable Test System (PCTS) and two hardware configurations at LBNL.  The list of tests on that page gives an idea of what is currently being tested.

OUTSTANDING ISSUES

  • Many things are not thoroughly tested yet, notably LC functionality, flashers, and programmable dead time. 
  • Compressed data has not been thoroughly examined.
  • A few key messages still missing from the documentation.
  • Request has been made for a rework of the monitoring to reduce overall bandwidth and increase relevance of monitoring information.
  • Should implement raw data format so that compressed data can be compared with uncompressed in a one-to-one format.

INTEGRATION STATUS

  • David Hays is currently testing the integration of domapp with the DOM Hub application.
  • Much more testing and use by physicists with experience looking at IceCube data (similar to FAT) is required!
